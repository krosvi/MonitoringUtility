MonitoringUtility
======================================================

What is it?
------------------------------------------------------
MonitoringUtility - it's an open souce utility for Microsoft Windows that show information about processes, services, autorun, active connections.

Features
------------------------------------------------------
View information about:

* Current processes
* Current services
* Autorun
* Active connections

Technology
------------------------------------------------------
* C#/.Net Framework
* WMI
